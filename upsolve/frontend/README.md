# Upsolve Frontend

Upsolve is a competitive programming analysis platform that helps developers track their coding progress, analyze performance metrics, and improve their problem-solving skills.

## 🚀 Features

- **Profile Analysis**: Comprehensive analysis of coding profiles with detailed statistics
- **Progress Tracking**: Visual heatmaps and charts to track your coding journey
- **Performance Metrics**: In-depth insights into submission patterns and success rates
- **Peer Comparison**: Compare your progress with other developers
- **Smart Recommendations**: Get personalized suggestions to improve your coding skills

## 🛠️ Tech Stack

- **Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS
- **UI Components**: Custom components with modern glass-morphism design
- **Charts**: React Calendar Heatmap for submission visualization
- **Styling**: Custom color palette with professional branding

## 🎨 Design System

Upsolve uses a custom color palette:
- Primary: `#5BC8AF` (Mint Green)
- Secondary: `#5FB0B7` (Teal Blue)  
- Accent: `#6C91BF` (Soft Blue)
- Highlight: `#A14EBF` (Purple)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
